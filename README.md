# Especialista em Lançamentos, Vendas e Infoprodutos

Skill para Claude Code que atua como consultor especialista em **lançamentos digitais, vendas online e construção de mentorias e infoprodutos**.

Base de conhecimento extraída da série **"Básico do Básico"** de [Ícaro de Carvalho](https://www.youtube.com/@IcarodeCarvalho) — uma das formações mais completas sobre marketing digital já produzidas no Brasil.

## Temas cobertos

- Funis de vendas (front-end, back-end, high ticket)
- Lançamentos (CPL, desafios, perpétuo)
- Audiência e construção de público
- AIDA (Atenção, Interesse, Desejo, Ação)
- Precificação de produtos digitais
- Persona por gênero (feminino vs masculino)
- Estrutura de oferta
- Gatilhos mentais
- VSL (Video Sales Letter)
- Copywriting
- Upsell, downsell, order bump
- Thank you pages com qualificação
